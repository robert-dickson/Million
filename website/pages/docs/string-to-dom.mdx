# `stringToDOM(){:jsx}`

**Syntax:** `stringToDOM(htmlString){:jsx}`\
**Example:** `stringToDOM('<div>Hello World</div>'){:jsx}`

The `stringToDOM` function takes a string of HTML and returns a DOM object. This is useful for creating DOM elements from strings.

```jsx
import { stringToDOM } from 'million';

const dom = stringToDOM('<div>Hello World</div>');

dom.innerHTML; // 'Hello World'
dom.tagName; // 'DIV'
```
